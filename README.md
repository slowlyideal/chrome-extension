# chrome-extension  
功能：实现当前网址的保存，重启浏览器后可以再打开，可以用来重启保存网址。  
使用：开启谷歌浏览器的开发者模式，下载chrome-extension.zip，加载解压后的文件夹即可   
效果演示：http://bilibili.com/video/BV1MK411P73F/
